# Field-Project: Hotel Orient Pearl

This repository contains the source code for the Hotel Orient Pearl website, a luxury multi-cuisine and premium stay destination.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Structure
- `FRONTEND/`: Contains the frontend assets, including HTML, CSS, images, and videos.
- `backend/`: Contains the backend logic (if any).

## Features
- Beachside ambiance gallery
- Multi-cuisine menu
- Event catering and celebrations
- Responsive design
